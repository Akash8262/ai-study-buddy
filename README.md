
# AI Study Buddy

## Project Objective
AI Study Buddy is an educational chatbot designed to help students with study tips, exam preparation, Python programming, Artificial Intelligence, and Machine Learning concepts.

## Features
- Study tips
- Exam preparation guidance
- Python learning assistance
- AI concept explanations
- Machine Learning explanations
- Fallback responses

## Technologies Used
- Botpress
- Generative AI
- NLP
